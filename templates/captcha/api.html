接口信息
属性 值 备注
接口地址 http://172.30.157.96:5000/captcha
返回格式 json
请求方式 POST
请求示例 http://172.30.157.96:5000/captcha?str="the base64Str of your image" str 的值为图片的 base64 加密后的字符串
返回参数说明
名称 类型 说明
data string 识别的结果
status int 返回码
message string 返回说明
JSON 返回示例
{
    "data":"ABCD"
    "status": 200
    "message":"success"
}
Python 示例代码
import json
import requests
import logging
import base64


url = 'http://172.30.157.96:5000/captcha?str=' + i['imageBase64']
r = requests.post(url)
result = json.loads(r.text)
for i in result:
    print("data:" + i[data])
    print("status:" + str(i[status])
    print("message:" + i[message])
